# Corpus Programming

A collection of tools, scripts, and examples for **Corpus Programming** — the computational analysis and processing of large text collections (corpora). This repository demonstrates techniques for loading, cleaning, analyzing, and extracting linguistic patterns from text datasets.

## Overview

Corpus programming focuses on working with large-scale textual datasets using programming techniques. This project provides utilities and examples for:

- Corpus preprocessing
- Tokenization and normalization
- Frequency analysis
- N-gram extraction
- Concordance generation
- Keyword analysis
- Linguistic pattern discovery

The repository is designed for **linguistic, and NLP researches**, corpus-based analysis.

## Features

- Text corpus loading and preprocessing
- Tokenization and normalization pipelines
- Word and phrase frequency statistics
- N-gram generation
- Concordance search
- Keyword-in-context (KWIC) analysis
- Modular scripts for extending functionality
